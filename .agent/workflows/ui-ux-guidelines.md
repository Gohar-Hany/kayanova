# UI/UX Design Guidelines

## Typography Scale
- H1: 2.5rem-3rem (page titles)
- H2: 2rem (section headers)  
- H3: 1.5rem (card titles)
- Body: 1rem
- Small: 0.875rem
- Tiny: 0.75rem

## Spacing (8-point grid)
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px

## Button Hierarchy
1. Primary: Filled gradient, bold
2. Secondary: Outlined/subtle
3. Tertiary: Text only

## Animation Timings
- Fast: 150ms
- Normal: 200-300ms
- Slow: 400-500ms

## Shadow Levels
- Level 1: 0px 1px 3px rgba(0,0,0,0.1)
- Level 2: 0px 4px 6px rgba(0,0,0,0.1)
- Level 3: 0px 10px 20px rgba(0,0,0,0.15)
- Level 4: 0px 20px 40px rgba(0,0,0,0.2)

## Accessibility
- Min contrast: 4.5:1
- Min touch target: 44x44px
- Always show focus indicators
- Support keyboard navigation
